## Configuration

1. Framework - [Angular7](https://angular.io) 
2. Authentication - [Auth0](https://auth0.auth0.com) ([@auth0/angular-jwt](https://www.npmjs.com/package/@auth0/angular-jwt))
3. UI - [fomantic-ui](https://fomantic-ui.com/) 
4. Server & Deploy - [zeit-now](https://zeit.co/)

## Usage

```
npm i
ng serve    # http://localhost:4200
```
